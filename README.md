# 🎧 Axtro - Landing Page

> Official landing page for Axtro, a premium headphone brand focused on high-quality sound and sleek design.

## 🌐 Live Demo

🔗 **[Click here to visit Axtro](https://otavioalvessilva.github.io/productPage/)**

## 📋 About The Project

**Axtro** is a landing page created to showcase the **M3 Plus** headphone line. The project was built with a focus on:

- ✅ Modern and minimalist design
- ✅ Smooth user experience (UX)
- ✅ Full responsiveness across all devices
- ✅ Accessibility and semantic HTML best practices

---

## 🚀 Technologies Used

- ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white) **HTML5** - Semantic and accessible markup
- ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white) **CSS3** - Styling with variables and Flexbox
- ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black) **JavaScript** - Interactivity and smooth navigation
- ![Git](https://img.shields.io/badge/Git-F05032?style=flat&logo=git&logoColor=white) **Git** - Version control
- ![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-222222?style=flat&logo=github&logoColor=white) **GitHub Pages** - Free hosting

---

## 🎨 Features

- **Fixed navbar** with hide-on-scroll-down / show-on-scroll-up effect
- **Mobile menu** with animated hamburger and intuitive icons
- **Smooth scrolling** between sections
- **Organized sections:**
  - Hero section with call-to-action
  - Technical specifications
  - Product differentiators
  - Available colors
  - Newsletter subscription
  - Footer with social media links

---

## 📱 Responsiveness

The layout adapts perfectly to different screen sizes:

| Device | Breakpoint |
|--------|------------|
| 📱 **Mobile** | up to 768px |
| 💻 **Tablet** | 768px to 1024px |
| 🖥️ **Desktop** | above 1024px |
